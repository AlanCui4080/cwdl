真实CW音频生成

比赛名的生成：从任意大洲的缩写名称或者DX或者TEST中抽取
呼号的生成：1-2个随机字母+一个数字+2-3个随机字母
序号的生成：0-999的数字 如果是一位数和两位数前面补零 譬如01 023


- 1-2个CQ 比赛名 呼号A 
上面的序列原样重复1-3遍 间隔20-100空格
- 随机间隔10-30个空格
- 呼号B
上面的序列原样重复1-5遍 长尾分布 1遍有70%概率 间隔10-30空格
- 随机间隔5-20个空格
- 呼号B 5NN 10%的概率出现NR 序号 50%的概率出现一个K
- 随机间隔5-20个空格
- TU 呼号A 5NN 10%的概率出现NR 序号 20%的概率出现一个BK
- 5% 概率出现2-3个间隔两空格的E
在上述序列中 序号和呼号有2%的概率出现错误，若出现错误则随机出错序列中的1-2个字母数字 [DEL3]到[DEL8]后拍法正确的序列

在上述序列中 第二句的呼号B有2%可能被抄错 如果抄错了 在抄错的序列后追加一个 QRZ 重复两遍呼号B 随机间隔5-20个空格 随后重发上一句
在上述序列中 第二句的呼号B有2%可能听不清 如果听不清 直接发送QRZ? 随后随机间隔5-20个空格 发送一到两遍呼号
譬如 CQ TEST BA1BA  CQ TEST BA1BA   BB2BB    BA2BB 5NN 03   QRZ BB2BB BB2BB      BB2BB 5NN 03 K    TU BA1BA 5NN 92 BK
